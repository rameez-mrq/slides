Cranfield



The Cranfield experiments were a series of experimental studied in information retrieval, to evaluate the efficiency of indexing systems. The dataset is available here.


  • 1400 Documents
  • 225 queries
  • Relevance file

trec_eval


trec_eval is the standard tool used by the TREC community for evaluating retrieval system, given the results file and a standard set of judged results.


  • Installation
    • Build from source.
  • Usage: 
                            trec_eval -m measure[.params] rel_info_file results_file
  • Format result file (ref)
